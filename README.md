# Pwnagotchi

This contains resources I used for my pwnagotchi. If you make one, feel free to use this stuff. Or not, do what you want lol.

It is a Slimagotchi.
- Waveshare Screen (desoldered connector)
- Pi Zero 2W with trimmed headers so waveshare screen sits flush
- PiSugar S (unsure if compatible with PiSugar products)
- 803160/803060 Battery (Upgrading PiSugar from 1200mAh to 2200mAh)

CAD files here are for the above spec. Original and 803060 battery should both work.
